This directory contains folders: data (solar, log, weather, UV-Vis), scripts, figures, writing.

Data: dirt data, in a hand-made csv file; 3 types of data automatically logged from solar field; UV-Vis raw data
Scripts: contains Mathematica notebooks for analysing UV-Vis data and solar field data (3 types). 
Writing: papers written on the soiling study; suggestions for future research

See README files located in each of the aforementioned folders for more details.
